# ocr-ss

Wayland screenshot tool with OCR overlay.

## Features
- Region screenshot (grim + slurp)
- OCR via Tesseract
- Visual highlight of detected text
- Copy all detected text
- Native GTK4 UI

## Requirements
- Wayland
- grim
- slurp
- wl-clipboard
- tesseract
- python-gobject
- gtk4

## Usage
```bash
ocr-ss
```
